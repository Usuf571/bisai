# 🎯 BiSAI - Командная викторина "Перетягивание каната"

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-Production%20Ready-brightgreen)

BiSAI - это веб-приложение для проведения интерактивных командных викторин с механикой "Перетягивание каната". Две команды (красная и синяя) отвечают на вопросы, и каждый правильный ответ перемещает виртуальный канат в сторону команды!

## 🎮 Как это работает?

```
КОМАНДНАЯ ВИКТОРИНА
┌─────────────────────────────────────────────────────────┐
│                                                          │
│  Красная команда        ?        Синяя команда         │
│      🚩←────────────────────────────→🚩               │
│       Марк        Правильный ответ!      Маша          │
│       Иван        Синяя команда тянет канат →           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Процесс игры:
1. 👤 **Регистрация и вход** в систему
2. 📝 **Выбор теста** или **создание своего**
3. 🎯 **Ввод имен команд** (Красная и Синяя)
4. 👥 **Добавление игроков** в каждую команду
5. ❓ **Ответы на вопросы** - каждый правильный ответ = +1 очко
6. 🏆 **Результаты** - команда с большим счетом побеждает!

- ✅ **Создание тестов** с поддержкой различных типов вопросов

- ✅ **Фоновая музыка** для атмосферы

## 🚀 Быстрое начало

### 1️⃣ Установка

```bash
# Клонировать репозиторий
git clone https://github.com/Usuf571/bisai.git
cd bisai

# Или просто распаковать архив
# unzip bisai.zip
# cd bisai
```

### 2️⃣ Подготовка БД

```bash
# Импортировать SQL схему
mysql -u root -p < database/migrations/001_create_initial_tables.sql

# Или используя phpMyAdmin (см. INSTALL.md)
```

### 3️⃣ Конфигурация

Отредактируйте `config/db.php`:

```php
return [
    'host' => 'localhost',
    'database' => 'bisai_db',
    'username' => 'root',
    'password' => '',
];
```

### 4️⃣ Запуск

```bash
# Встроенный сервер PHP
php -S localhost:8000 -t public/

# Или через Apache/Nginx
# Открыть http://localhost/bisai/public/
```

### 5️⃣ Создать первый тест

1. Зарегистрируйтесь на сайте
2. Перейдите в личный кабинет
3. Нажмите "Создать тест"
4. Добавьте вопросы и ответы
5. Опубликуйте тест
6. Выберите его в главном меню для игры

## 📝 Примеры использования

### Создание теста программно

```php
<?php
require_once 'includes/autoload.php';

// Создать новый тест
$test = new Test();
$test->attributes = [
    'creator_id' => 1,
    'title' => 'Математика для начинающих',
    'description' => 'Базовые операции сложения и вычитания',
    'difficulty' => 'easy',
    'status' => 'draft',
];
$test->save();

// Добавить вопрос
$question = new Question();
$question->attributes = [
    'test_id' => $test->id,
    'question_text' => 'Сколько будет 2 + 2?',
    'question_type' => 'multiple_choice',
    'order' => 1,
];
$question->save();

// Добавить варианты ответов
$question->addAnswer(['answer_text' => '3', 'is_correct' => false, 'order' => 1]);
$question->addAnswer(['answer_text' => '4', 'is_correct' => true, 'order' => 2]);
$question->addAnswer(['answer_text' => '5', 'is_correct' => false, 'order' => 3]);

// Опубликовать тест
$test->publish();
?>
```

### Запуск игры программно

```php
<?php
// Создать игровую сессию
$session = GameSession::createNew(1, 'Красная команда', 'Синяя команда');

// Добавить игроков
$session->addPlayer('Алиса', 'red', $userId1);
$session->addPlayer('Боб', 'blue', $userId2);

// Начать игру
$session->start();

// Получить текущий вопрос
$question = $session->getCurrentQuestion();

// Перейти к следующему
$session->nextQuestion();

// Завершить игру
$session->finish();

// Получить результаты
$results = $session->getResults();
?>
```
